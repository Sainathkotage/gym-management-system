
💪 Gym Management System
==========================

A web-based Gym Management System designed to digitize the way gyms handle billing, communication, and user management.
This project is built using HTML, CSS, JavaScript, Firebase, and enhanced with TailwindCSS for a clean, modern user interface.

--------------------------

🚀 Features:
- Role-Based Authentication (Admin, Member, User)
- Digital Billing System using Firebase Firestore
- Notification System for fee reminders
- Admin Panel to manage members and generate bills
- User and Member Dashboards
- Real-Time Data Handling

--------------------------

🛠️ Tech Stack:
- Frontend: HTML, CSS, JavaScript
- Styling: TailwindCSS
- Backend/Database: Firebase (Auth + Firestore)
- Design: Figma

--------------------------

📂 Project Modules:
- Admin Panel: Manage members, bills, notifications
- Member Panel: View personal bills and notifications
- User Panel: View gym information and explore plans

--------------------------

🔧 How to Run:
1. Clone the repository:
   git clone https://github.com/Sainathkotage/gym-management-system.git
   cd gym-management-system

2. Install TailwindCSS (if using React setup):
   npm install
   npm run dev

3. Set up Firebase:
   - Go to https://console.firebase.google.com/
   - Create a project
   - Enable Firestore and Authentication
   - Replace config in firebaseConfig.js
